## <Icon name="open-book" class="inline-icon" label="Interpret code" /> Tolka kod

1. Vad gör följande kod och vad blir utskriften?  
    ```python
    for i in range(2, 10, 2):
        print(i)
    ```

2. Vad blir resultatet av koden nedan och varför?  
    ```python
    for i in range(2):
        for j in "hej":
            print(i, j)
    ```

4. Vad är syftet med att fånga ett exception som en variabel så här, `e`:  
    ```python
    except ValueError as e:
        print("Fel:", e)
    ```

5. Titta på nedanstående kod och svara på följande frågor utan att exekvera koden:  
    - Vad skrivs ut när programmet körs?
    - Förklara varför loopen slutar.
    - Vilka värden får **y** under loopens gång?
    ```python
    x = 10
    y = 0

    while x > 0:
        if x % 2 == 0:
            y += x
        else:
            y -= x
        x -= 3

    print("Slutvärde av y är:", y)
    ```

