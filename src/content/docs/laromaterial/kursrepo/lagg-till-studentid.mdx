För att våra integrationer med Canvas ska fungera behöver du lägga till ditt Canvas student id som en variabel i ditt GiHub repo. 

## Kopiera Canvas student ID

Börja med att gå till Canvas, [canvas.bth.se](https://canvas.bth.se), och denna kurs. När du är inne på kurssidan gå till "Personer"/"People". Sök sedan på ditt namn och klicka på dig själv i listan. **PS** det är viktigt att du väljer dig själv annars blir inte ditt betyg rapporterat till Canvas.

<Figure 
    src={canvas1}
    caption="Hitta dig själv på Canvas"
/>

Om det finns fler med samma namn som dig, som det gör för mig i bilden. När du klickar på namnet ska det finnas en knapp, "Edit Profile", om det är din användare. Om den knappen inte finns är det fel användare.

När du är inne på din profil, kopiera de sista siffrorna i URL:en, efter `/users/`, det är ditt Canvas student ID. För mig är det `32763`. Kopiera ditt!

import canvas2 from '@assets/kursrepo/canvas2.png';

<Figure 
    src={canvas2}
    caption="Kopiera Canvas Student ID"
/>

## Skapa variabel i GitHub med ditt ID

Gå till ditt GitHub repo för kursen. Klicka sen på `Settings` --> `Secrets and variables` --> `Actions` och `New repository variable`.

import canvas3 from '@assets/kursrepo/canvas3.png';

<Figure 
    src={canvas3}
    caption="Gå till New repository variable."
/>

Klistra in ditt ID i rutan för `value` och ge den namnet `STUDENT_CANVAS_ID`.

<Figure 
    src={canvas4}
    caption="Nu är du på rätt plats i filsystemet."
/>

Nu är du redo att börja jobba med ditt kursrepo.
