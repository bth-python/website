För att kunna arbeta med kursens kod och uppgifter på GitHub behöver du koppla ditt GitHub-konto till kursens organisation. 

På kursens [Canvas](https://canvas.bth.se) sida finns en uppgift som heter `Koppla GitHub konto till organisationen`. Gå till den och lämna in uppgiften. Din inlämning ska vara den mejladress som du har kopplat till ditt GitHub-konto.

import organisation from '@assets/kursrepo/organisation.png';

<Figure 
    src={organisation}
    caption="Lämna in uppgiften med din GitHub mejladress."
/>

Inom 3 minuter kommer uppgiften automatiskt rättas och du får en inbjudan till organisationen. Du kommer att få en notis i Canvas och ett mejl till den mejladress du angav i inlämningen.

import organisation2 from '@assets/kursrepo/organisation2.png';

<Figure 
    src={organisation2}
    caption="Du får inbjudan som en kommentar på inlämningen."
/>

När du har fått inbjudan kan du acceptera den genom att klicka på länken i mejlet eller i Canvas.

import organisation3 from '@assets/kursrepo/organisation3.png';

<Figure 
    src={organisation3}
    caption="Acceptera inbjudan till organisationen på GitHub."
/>

## Aktivera Single Sign-On (SSO)

:::Note
Om knappen `Single Sign-On` inte syns kan hoppa över detta steg.
:::

När du har accepterat inbjudan till organisationen behöver du aktivera Single Sign-On (SSO) för att kunna arbeta med kursens kod och uppgifter. Klicka på knappen `Single Sign-On` uppe till höger på sidan i organisationen.

import organisation4 from '@assets/kursrepo/organisation4.png';

<Figure 
    src={organisation4}
    caption="Acceptera inbjudan till organisationen på GitHub."
/>

När du har aktiverat SSO är du redo att arbeta vidare.
