[GitHub Actions](https://github.com/features/actions) är ett verktyg som används för att automatisera arbetsflöden i GitHub. Det kan användas för att köra tester, bygga och distribuera kod, och mycket mer. I den här kursen kommer vi att använda GitHub Actions för att rätta din kod vid inlämning. Du behöver inte lära dig något om det i kursen. Det är bara en fil som måste finnas i ditt repo för att rättningen ska fungera.



## Ladda ner workflow-filen

För att ladda ner workflow-filen som används för att rätta din kod, använd `uv` och Task. Denna fil kommer att placeras i katalogen `.github/workflows/` i ditt kursrepo.

```console title="Ladda ner workflow-filen"
uv run task download-workflow
```

Nu har du fått katalogen `.github/workflows/` i ditt kursrepo. I den mappen finns filen `examiner.yml`. Om du står i kursrepo mappen med terminalen kan du se innehållet i katalogen med kommandot:

```console title="Visa innehållet i katalogen"
ls -l .github/workflows/
```

Gör en commit och pusha katalogen.
