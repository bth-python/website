# Listor

Denna del är under arbete och är inte klar.
:::

I denna övningen får du träna på att programmera med listor och några inbyggda metoder.

## Förutsättning

Du har jobbat igenom materialet i kmom04.

## Förberedelser

Förbered dig så här för övningen. Börja med att öppna din terminal och gå till ditt kursrepo.

```
cd dbwebb-kurser/python-<din akronym>
```

Om du inte har startat VS Code än kan du göra det nu.

```
code .
```

Flytta sen ner till lab_04 katalogen för att ladda ner filerna som behövs för labben.

```
cd src/kmom04/lab_04
```

Nu skall vi hämta labb-filerna från nätet som behövs för att komma igång med labben. Skriv följande kommandon.

```
uv run task download-lab -- lab_04
```


Exekvera labben och se att den fungerar.

```
uv run lab.py
```

Sätt igång och lös uppgifterna i labben.
